# Classic Bingo

A feature-rich, cross-platform Bingo game built with Flutter.

## ?? Live Demo
Check out the web version here: [https://classic-bingo.vercel.app](https://classic-bingo.vercel.app)

## ? Features
- **Multi-platform support**: Android, iOS, Web, Windows, macOS, and Linux.
- **Online & Offline Modes**: Play by yourself or join rooms with friends.
- **Real-time Synchronization**: Built with Firebase for seamless multiplayer gameplay.
- **Beautiful UI**: Modern design with smooth animations and intuitive navigation.
- **Customizable Settings**: Adjust game rules and host your own rooms.

## ??? Tech Stack
- **Framework**: [Flutter](https://flutter.dev)
- **State Management**: Provider/ChangeNotifier
- **Backend**: Firebase (Core & Realtime Database)
- **Deployment**: Vercel (Web)

## ?? Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/BhairavJShah/classic-bingo.git
   cd classic-bingo
   ```

2. **Install dependencies:**
   ```bash
   flutter pub get
   ```

3. **Run the app:**
   ```bash
   flutter run
   ```
